Example App

This simple "Hello, World" app is built by Minipaca. It runs fast on EndeavourOS and takes about 180MB RAM.


Create App in 4 Steps

Step 1: Install Minipaca

Minipaca packages here.

Step 2: Create Source Code

$ minipaca -x app-gtk4
$ cd app-gtk4

Step 3: Compile

$ minipaca -g

Step 4: Pack

$ minipaca -p

The packages of "Hello, World" app are in directory 'p'.
