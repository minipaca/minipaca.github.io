Tips for Windows OS users:
1. The executable file is a PowerShell script: 'C:\Program Files\minipaca\bin\minipaca.ps1'
2. If it is not allowed to run script in PowerShell, you can setup it with command: Set-ExecutionPolicy RemoteSigned
3. Download NSIS and install it for packing feature
